Tracksmith

A JSON RESTful API for the Irish Rail Realtime API



What is Tracksmith?

Tracksmith is a RESTful API which pre-parses Irish Rail's Realtime API to provide a simpler response that is easier to use for a multitude of projects.
It also provides some features which are not available in the Irish Rail API, such as the ability to get network-wide or route-specific delays, the routes of a train, and the stations on a route.



Documentation

All Tracksmith endpoints are accessible only through POST requests. A successful request will return a JSON object with the requested data, and a status of 200.
Errors return a JSON object with an error status and a message.
Tracksmith endpoints are split into three sections,

  • Stations
  • Trains
  • Other

Press on one of the buttons below to open the relevant section.

Stations

Stations is split into five sections, all, by code, by name, by type, and info.
